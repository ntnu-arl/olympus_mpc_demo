enum olympus_base_joint

Enum representing the types of base joints for the Olympus robot.

Enumerator

A fixed joint with no degrees of freedom.

A revolute joint around the x-axis. State is [q,dq/dt]

A revolute joint around the y-axis. State is [q,dq/dt]

revolute_z 

A revolute joint around the z-axis. State is [q,dq/dt]
